# Stock Market Prediction using LSTM

This project predicts future stock prices using Deep Learning techniques. 
The model is built using LSTM (Long Short-Term Memory) networks and trained on historical stock market data collected from Yahoo Finance.

## Features
- Real-time stock data using yfinance
- Data preprocessing and normalization
- LSTM deep learning model
- Future stock price prediction
- Data visualization using Matplotlib

## Technologies Used
- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- yfinance

## Project Goal
To analyze historical stock data and predict future market trends using AI and Deep Learning techniques.
